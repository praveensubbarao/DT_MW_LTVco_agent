# DT_MW_LTVco_agent — Claude Code Instructions

## Project

Production-grade Playwright + TypeScript E2E testing framework for lifetimevalue.co with an AI-powered test generation agent.

**Stack:** Playwright 1.40 · TypeScript 5.3 · Node 20+ · Anthropic SDK

## Key Conventions

- Every `test()` block **must** have a `// Steps:` comment immediately above it (enforced by `npm run precommit`)
- Use the `appActions` fixture from `@/utils/fixtures/appBaseTest` — never import `page` directly in tests
- Tests live at `src/tests/{featureName}/{featureName}.spec.ts`
- Path alias `@/*` maps to `src/*`

## Device Profiles

All tests run across 4 responsive profiles (configured in `playwright.config.ts`):

| Profile | Dimensions | Device |
|---------|-----------|--------|
| `desktop-chrome` | 1350×940 | Desktop |
| `tablet-chrome` | 1024×1366 | iPad Pro |
| `mobile-chrome` | 393×851 | Pixel 5 |
| `mobile-small` | 390×844 | iPhone 12 |

## Environment

```env
STACK=dev|stg|prod        # Default: prod → www.ltvco.com
LLM_PROVIDER=claude       # ollama | openai | claude
ANTHROPIC_API_KEY=...     # Required for Claude provider and test generation
```

Base URLs resolve from `STACK`:
- `prod` → https://www.ltvco.com/
- `stg` → https://stg.ltvco.com/
- `dev` → https://dev.ltvco.com/

## Test Generation (testOrchestration)

Generate Playwright tests from feature descriptions using Claude.

**Via Claude Code skill** — invoke `/testOrchestration` in the chat.

**Via CLI:**
```bash
echo "Feature: Homepage
Description: Test homepage loads correctly
Steps:
  1. Navigate to homepage
  2. Verify page title is visible
  3. Verify main navigation exists" | npm run generate:test
```

Full details in [agent/README.md](agent/README.md).

## Common Commands

```bash
npm test                          # All 4 device profiles
npm run test:desktop              # Desktop only
npm run test:tablet               # Tablet only
npm run test:mobile               # Mobile (Pixel 5)
npm run test:mobile-small         # iPhone 12
npm run test:headed               # With visible browser
npm run test:retry-failed         # Retry last failures
npm run generate:test             # Generate test from feature context
npm run precommit                 # Validate // Steps: comments exist
npx playwright show-report        # Open HTML report
STACK=dev npm test                # Run against dev environment
```

## AppActions Fixture API

| Method | Description |
|--------|-------------|
| `openPage(path?)` | Navigate + wait for networkidle |
| `clickLocator(locator)` | Click with 3-tier fallback |
| `expectPageBodyContains(text)` | Assert text in body |
| `expectPageUrlContains(fragment)` | Assert URL fragment |
| `getSelfHealingLocator(description, primary)` | LLM-assisted healing |
